# Osama Alzain - Official Website

Official website for Osama Alzain, featuring music, gallery, and contact information.

## Features

- Bilingual support (Arabic/English)
- Responsive design
- Music video player
- Photo gallery
- Social media integration
- Contact information

## Deployment on Vercel

This project is configured for easy deployment on Vercel.

### Quick Deploy

1. Push your code to GitHub/GitLab/Bitbucket
2. Import the repository in [Vercel](https://vercel.com)
3. Vercel will automatically detect the static site configuration
4. Click "Deploy"

### Manual Deployment

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

## Local Development

```bash
# Install dependencies (if any)
npm install

# Run local server
npm run dev
```
